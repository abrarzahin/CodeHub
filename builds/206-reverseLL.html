206-reversing Singly Linked list using Iteration


Reverse a singly linked list

Example:

Input: 1->2->3->4->5->NULL
Output: 5->4->3->2->1->NULL 

        Solution:
        public ListNode reverseList(ListNode head) {
            ListNode prev = null;
            ListNode curr = head;
            while(curr!=null){
                ListNode nextTemp = curr.next;
                curr.next = prev;
                prev = curr;
                curr = nextTemp;


            }
            return prev;
        }
